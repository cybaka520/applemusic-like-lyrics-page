# AMLL Player Web


##  使用说明

### 输入方式

1. **本地文件上传**
   - 音频文件
   - 歌词文件
   - 封面图片（可选）

2. **URL参数**
   ```
   ?music=音乐链接&lyric=歌词链接&cover=封面链接&title=歌曲名&artist=艺术家
   ```
